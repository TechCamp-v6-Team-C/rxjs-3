## Ejecutar el proyecto.

Cuando descargue el código ejecute el comando:

``` npm install ```

Para instalar los módulos necesarios para el proyecto.

Una vez se instalen todos los módulos ejecute el comando:

``` npm start ```

Para levantar el servidor.
